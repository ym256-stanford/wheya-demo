# MeetingUp (iOS)

This repository contains the code for a meeting scheduling iOS app for **demonstration purposes**. The app helps you create and manage meetings, and (where supported in the build) view participantsâ€™ **real-time locations** and their **estimated time of arrival (ETA)**.

## Features
- User authentication flow (Register / Login / Sign out)
- View a list of upcoming meetings
- View meeting details
- Create, edit, and delete meetings
- Real-time participant location + ETA display (when enabled/configured)

## Requirements
- macOS with **Xcode**
- iOS Simulator or a physical iPhone
- **Apple ID signed into Xcode** (required to run on a device and often needed for signing)
